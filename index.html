Pflanzprojekt

Gründe/Probleme

Ich hatte mir überlegt, dass eine automatische Pflanzenbewässerungsanlage z.b. für Fassaden eine gute Idee wäre. Dann hatte ich online nur teuer Lösungen gefunden, außerdem habe ich Lust am Experimentieren.

Nachbauanleitung

Raspberry Pi imiger installertion + Homeassistant insterllation

config yamel esphome


esphome:

  name: pflanzprojekt

  friendly_name: Pflanzprojekt

#verwendetes Bord

esp32:

  board: esp32dev

  framework:

    type: arduino

#Arduinino ide

# Enable logging

logger:

web_server:

  port: 80

  css_include: ./0.css

# Aktiviren von Home Assistant API für alle die Home assistant nutzen

api:

  encryption:

    key: "Euer homeassistant key wenn man das adonn Esphome insterliert vunktioniert es automatisch"

# Dratlossen Update in den Code schreiben

ota:

  password: "Euer Passwort"

# ssid Wlan name

# Wlan Password

wifi:

  ssid: !secret wifi_ssid

  password: !secret wifi_password

  # Enable fallback hotspot (captive portal) in case wifi connection fails

  ap:

    ssid: "Pflanzprojekt Fallback Hotspot"

    password: "Pflanzprojekt007"

# dht adafrut Bibliotek temperatur

captive_portal:

sensor:

  - platform: dht

    pin: 5

    temperature:

      name: "Pflanzentemperatur"

    humidity:

      name: "Pflanzenluftfeuchtigkeit"

  - platform: internal_temperature

    name: "Internal Temperature"

# switch darf nur einmalgenant werden

# Relai (für die Pumpen)

switch:

  - platform: gpio

    name: "Bewässerung"

    pin: 13

  - platform: gpio

    name: "Bewässerung2"

    pin: 14
